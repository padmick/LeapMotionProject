Description:

Generated with the yo man kiwi kit and leapControler plugin this app allows users to control the character on the screen via the leap motion connection.

Inital verions of this application take data from the leap motion and mapped it to d3.js those seemed clumbersome and so it became easier to create something using a gaming engine in the end.
And so here is a simple rendition of flappy bird mapped to a leap motion controller.
